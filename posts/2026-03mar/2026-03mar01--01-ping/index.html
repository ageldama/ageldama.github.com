🧑🏽‍💻 Unicode 17.0 Emoji 목록 자동화 + Claude AI

2026년03월01일
Claude을 이용하여 이모지 목록을 Unicode 17.0 Emoji 목록에서 자동으로 만들도록 해봤다

문제, 의도 💣

제목에 자동으로 emoji 넣기-을 계속 해왔는데, 이 emoji목록은 손으로 수집해서 만든거였고, 가끔 Unicode-에는 있지만, 이 수동으로 만든 목록에는 없는 경우가 있어서 아쉬웠다.

그래서 이번엔 이걸 고쳐보기로 했다.

Claude 🤖: 자동으로 다운로드 => 모든 이모지를 출력

그래서 유니코드 DB에 공개된 데이터파일들을 다운로드 받고, 거기에서 도출 가능한 모든 이모지를 출력하도록 했다.

생각보다 텍스트파일을 파싱하고 하는 스크립트가 귀찮았다. 예전에 유니코드<=>CP949 변환이나 조합형 한글코드 변환에도 이런 파싱+출력, 혹은 코드매핑 테이블 생성하는 스크립트를 짜왔었지만 이번엔 조금 다르게 해볼까 했다.

그래서 Claude 사용하여 코드를 생성해봤다. 프롬프트에 적당히 원하는걸 입력해서 스크립트를 만들어봤다. 만족스러웠다. 그 다음에 다운로드 받아서 조금 더 내 목적에 알맞게 수정할 수 있었다. 1~2분 정도 생성해주기를 기다리는거 이외엔 별로 오래 걸리지도 않았다. 만족스러웠다.

이전에도 몇 가지 스크립트나 함수, 혹은 작은 프로그램이나 도구들을 이렇게 생성하여 만들어 써왔는데 점점 더 마음에 들어간다.

실제로는 Claude은 Sandbox안에서 실행하니 완벽히 동작하지는 않을 가능성이 있지만, 그래도 최선을 다해서 실행해줬다. 물론 다운로드하여 다시 실행하면서 고쳐나갔다.

  use Data::Section::Simple qw(get_data_section);
  use Template;

  {
    my @emojis = ();

    while(<STDIN>) {
      my $ch = $_;
      chomp $ch;
      push @emojis, $ch;
    }

    my $tt = Template->new({
      INTERPOLATE  => 0,
    }) || die "$Template::ERROR\n";

    my $vars = {
      emojis => \@emojis,
    };

    my $tmpl = get_data_section('output.html');
    my $s = '';
    $tt->process(\$tmpl, $vars, \$s) || die $tt->error(), "\n";

    print $s;
  }

  __DATA__

  @@ output.html
  {{- $emojis := slice
      [% FOREACH emoji IN emojis -%]
      "[% emoji %]"
      [% END -%]
  -}}
